# Copilot-Repos

A collection of sample code demonstrating various programming concepts and patterns.

## Sample Code Files

### Python Sample: calculator.py
A simple calculator module demonstrating:
- Basic arithmetic operations (add, subtract, multiply, divide, power)
- Function documentation with docstrings
- Error handling for edge cases
- Example usage in main block

**Run:**
```bash
python calculator.py
```

### JavaScript/Node.js Sample: server.js
A basic HTTP server demonstrating:
- Creating an HTTP server with Node.js
- Routing and request handling
- JSON responses
- Environment variable configuration
- Health check endpoint

**Run:**
```bash
node server.js
```

### Java Sample: UserManager.java
A user management system demonstrating:
- Class-based OOP design
- CRUD operations (Create, Read, Update, Delete)
- Use of Java Collections (ArrayList)
- Optional type for null safety
- Inner class implementation
- Stream API usage

**Compile and Run:**
```bash
javac UserManager.java
java UserManager
```

## Purpose

This repository serves as a reference for common programming patterns and best practices across multiple languages.
